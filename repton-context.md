FLAGSHIP PRODUCT: AIOS (AI Operating System)
AIOS is Repton Digital's hero offering. It is the most ambitious and differentiated product.
What it is:
A fully custom-built AI system that connects an AI agent to your business's knowledge base, tools, and workflows — accessible via Telegram, operating 24/7.
The 5 layers of AIOS:

Central Knowledge Base — Every piece of information about the business (services, pricing, clients, SOPs, FAQs) fed into a single AI brain
AI Agent — One agent that has access to all knowledge and can act on it (answer questions, draft emails, look up info, trigger automations)
Telegram Interface — Business owner can talk to the agent via Telegram from their phone, anytime
Automation Layer — The agent can trigger real-world automations (send an invoice, update CRM, book a job, etc.)
Integrations — Connected to the tools the business already uses

What it's NOT:

Not a chatbot on a website
Not a generic AI tool like ChatGPT
Not a plug-in or template

It IS:

A bespoke, proprietary operating system for one specific business
Trained entirely on that business's data and processes
An agent that can actually do things, not just answer questions

Key message: "You run your business. AIOS runs everything else."
CTA for AIOS: "See AIOS in Action" → book a demo

DESIGN NOTES FOR DEVELOPERS

The brand is dark-first. Light mode is not a priority.
Glass UI is non-negotiable — every card, nav, and modal uses the liquid glass treatment.
The site should feel like something Apple would build if they were an AI agency.
Animations should be smooth, purposeful, and premium — not flashy for the sake of it.
Mobile must be excellent. The target audience (tradies) often browse on phone.
No stock photos. Use geometric/abstract 3D visuals, gradients, and UI elements only.
The word "Repton" in text should always be followed by "Digital" — never standalone.
"RPT." or "RPT.DIGITAL" is acceptable as a condensed logo mark.
Accent colors: sky blue (#0ea5e9) primary, soft indigo (#818cf8) secondary.
Never use purple-on-white gradients — it's overdone in AI design.
The founder is 16. The site should NOT feel like it was built by a 16-year-old. It should feel like it was built by a world-class agency.
